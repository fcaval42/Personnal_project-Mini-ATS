# Personnal_project-Mini-ATS

This is a personnal project in Python

The goal is to develop a Python application that can manage the tracking of applications in a recruitment process.
For now, the application runs from the command line, but I would like to create a web version. 

 ==== General constraints ====

- Python 3.x
- Object-oriented programming required
- Proper exception handling
- Code structured across multiple files
- No global variables
- Persistent data storage (JSON minimum)

 ==== Error handling ====

- A candidate must have a unique, auto-generated ID.
- It is not possible to assign an invalid status.
- Dates must be in a valid format.
- All errors must be handled properly (no crashes).

 ==== Mandatory features (MVP) ====
 
-- Candidate management --
Each candidate must include:

  -Unique ID
  -Last name
  -First name
  -Position sought
  -Application date
  -Status
  -HR notes

Possible statuses:
  -Received
  -Pre-qualified
  -Interviewed
  -Rejected
  -Accepted
